{{'Change Lock Settings' | translate}}
After the first timeout A in seconds, Canoe is locked using selected lock type. Choosing None disables locking after timeout A.
After the second timeout B, Canoe will lock and demand password to unlock. You can not disable this timeout, but you can set it very high.

When Canoe goes to background, this is the lock type used when coming back to foreground. Disable background locking by selecting None.
