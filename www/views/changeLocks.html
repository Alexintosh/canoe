{{'Change Lock Settings' | translate}}
After no user activity for soft timeout in seconds, Canoe is locked using selected lock type. Choosing None disables locking after soft timeout.
After additional hard timeout in seconds, Canoe will lock and demand password to unlock. You can not disable this timeout, but you can set it very high.

When Canoe goes to background, this is the lock type used. Disable background locking by selecting None.
